# ⏰ Clock

![Windows](https://img.shields.io/badge/Windows-10%20~%2011-blue)  
⏰ **シンプル時計アプリ** for Windows 10 / 11 / Android  
🌱 **メモリ使用量 90〜100MB / CPU数％で動作！**  
📦 シンプル・高速な時計 

---

## ✅ 特徴

- ✅ **軽量設計**（メモリ90〜100MB！）
- ⚡ **高速起動＆動作**
- 🖥️ **シンプルUI**
- 🎨 Windows～Androidまで幅広く対応

---

## 💻 対応環境

| OS              | 対応状況        |
|-----------------|----------------|
| Windows 10 (Version 2004,Build 19041)      | ✅ 対応        |
| Windows 11      | ✅ 対応確認済み |
| Android 16      | ✅ 対応確認済み |

> ※ ~8.1, 10(v2004 以下) は非対応です  
> ※ Android 15 以下は起動するか分かりません (Android 5 までは動くと思います)
